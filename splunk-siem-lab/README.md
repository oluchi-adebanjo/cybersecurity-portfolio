# Splunk SIEM Lab

Live threat detection using Splunk Enterprise and Windows Event Logs.

## What I did
- Installed and configured Splunk Enterprise on a Windows endpoint
- Ingested live Windows Security, System and Application Event Logs
- Simulated 3 MITRE ATT&CK attack techniques
- Detected all 3 attacks using custom SPL queries
- Documented findings in a formal Incident Investigation Report

## Techniques
- T1136.001 - Create local user account
- T1098 - Add account to Administrators group  
- T1053.005 - Create scheduled task for persistence

## Tools
Splunk Enterprise (Free), Windows Event Logs, PowerShell
